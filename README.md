# weight-tracker

体重管理PWA。摂取カロリー実績と体重トレンドの差分からTDEEを逆算し、週次で目標摂取カロリーを自動調整する個人用ツール(設計の詳細は `weight-tracker-design.md`、実装方針は `CLAUDE.md` を参照)。

## 運用フロー

1. あすけんに食事を記録する(通常運用)。
2. 1日の終わりに、あすけんのスクショ(食品名+kcal一覧)をClaudeチャットに渡し、P/F/Cと合計kcalを算出してもらう。
3. 算出結果と朝の体重を本アプリの「今日」タブに転記して保存する(TDEE逆算・週次目標見直しは自動で行われる)。
